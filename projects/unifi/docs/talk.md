## Setup

To check if UniFI Talk has been setup a GET request is sent to the url `<controllerIP>/proxy/talk/api/setup_complete`

The response is:

```json
{"setup_complete":false}
```

```bash
curl -k -XPOST -H 'Cookie: __stripe_mid=6a2f310a-edd5-4f3d-8291-294bbaf5993bee0eb3; _ga_5VPR3EBECJ=GS1.1.1727887746.1.1.1727889376.0.0.0; _ga=GA1.1.2084584409.1727887746; TOKEN=<REDACTED>' -H "Content-type: application/json" 'https://10.0.0.1/proxy/talk/api/setup_complete'
```
